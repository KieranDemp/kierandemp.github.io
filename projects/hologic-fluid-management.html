Hologic Fluid Management System

Hysteroscopic Surgery Fluid Management System

📅 2024-2025
🏥 Medical Device
🏆 University of Colorado Boulder

Project Overview

This project involved designing and building a closed-loop syringe pump system for hysteroscopic surgery applications. The system uses real-time pressure feedback and advanced PID control to improve flow stability compared to traditional peristaltic pump systems.

The device is designed to support minimally invasive procedures that treat uterine fibroids, affecting 80% of women. By replacing traditional peristaltic pumps with a dual-syringe system, we achieved better control over fluid flow and pressure during surgical procedures.

Background & Problem Statement

Approximately 80% of women develop uterine fibroids or polyps in their lifetime. Traditional treatments involve invasive surgery. Our device supports minimally invasive hysteroscopic procedures which are faster, have better patient outcomes, and shorter recovery times.

Design Challenge

Traditional fluid delivery systems use peristaltic pumps, which create significant pressure oscillations that can compromise surgical precision. We needed to design a system that:

  • Maintains stable intrauterine pressure (40-250 mmHg)
  • Reduces flow and pressure oscillations to ≤50% of baseline
  • Allows adjustable flow range from 0-700 mL/min
  • Provides real-time data display for surgical team
  • Uses disposable components for patient safety

System Design

Mechanical Hardware

The system uses a dual-syringe design with synchronized motion to deliver smooth, continuous fluid flow:

  • Syringe Drivers - Translates rotational motor motion into linear syringe movement
  • Mount Plates - Top and bottom plates ensure internal alignment for smooth motion and reduced friction
  • Motor Control - Stepper motors coordinate the actuation of both syringes for uniform pressure profile
  • Pressure Measurement - Flexible silicone diaphragm with integrated pressure sensor provides real-time feedback

Control Systems

The control system uses a closed-loop feedback mechanism to maintain stable pressure:

  • User Input - Surgeon sets target pressure on device interface
  • Motor Actuation - Control algorithm commands motor speeds
  • Pressure Feedback - Real-time sensor readings drive correction
  • PID Control - Tuned controller adjusts flow to maintain target pressure
  • Data Display - Real-time pressure, flow, and volume tracking on screen

Syringe Actuation Pattern

The motor control algorithm coordinates syringe movement to create a smooth pressure profile. Rather than single syringe operation, the dual-syringe system uses an overlapping actuation pattern where one syringe begins moving as the other completes its stroke, creating near-constant flow delivery.

Testing & Results

Specification Verification

We conducted comprehensive testing to validate system performance:

  • Flow Rate Test - Verified accurate delivery across 0-700 mL/min range
  • Pressure Test - Confirmed stable operation within 40-250 mmHg requirements
  • Volume Test - Validated ability to track inflow volume within ±25 mL up to 5L fluid

Dynamic Cavity Test

We developed a dynamic cavity test that simulates real surgical conditions. This test includes:

  • Simulated uterine cavity with variable compliance
  • Outflow valve that opens and closes like surgical conditions
  • Pressure monitoring that captures realistic surgical scenarios

Performance Comparison

Our pressure profile comparison demonstrated significant improvements over the legacy fluent system:

  • Reduced pressure oscillations by 70%
  • Maintained steady 150 mmHg target pressure with <5% deviation
  • Smoother response to valve opening/closing events

Key Achievements

  • Designed and prototyped complete fluid delivery system from concept to working prototype
  • Developed and tuned PID controller achieving 70% reduction in pressure oscillations
  • Implemented real-time sensor data processing with CRC validation
  • Conducted comprehensive testing validating all system requirements
  • Collaborated with Hologic and medical professionals throughout design process
